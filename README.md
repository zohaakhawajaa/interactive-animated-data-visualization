# Interactive Animated Data Visualization

## 📌 Project Overview
This project focuses on building animated and interactive data visualizations using Plotly Express.

The dataset used in this project is a hybrid dataset:
- Inspired by real-world socioeconomic data patterns
- Combined with custom-generated structured data created for deeper experimentation and learning

The goal was not just visualization — but data storytelling through motion and interaction.

---

## 🚀 Key Features
- Animated Scatter Plots (Time-based transitions)
- Animated Bar Charts
- Choropleth Maps
- Interactive hover insights
- Clean styling and readable layouts
- Structured data preprocessing

---

## 🛠️ Tech Stack
- Python
- Pandas
- Plotly Express
- Jupyter Notebook / VS Code

---

## 📊 Visualizations Included
1. GDP vs Life Expectancy (Animated Scatter)
2. Population & Growth Trends (Animated Bar Chart)
3. Country-Level Development Map (Choropleth)
4. Multi-variable interactive comparisons
